2026-07-04 - honesty fixes

The gate stops shipping broken pages

  • A generated page can no longer ship with a placeholder image or an empty-subject sentence — both used to slip through silently on a bad model response.
  • Flow-diagram edge labels now stagger so converging edges never overlap into unreadable text.
  • Page-spec rendering moved to a validated JSON contract (glance-catalog-001) instead of raw generated HTML, closing off a class of malformed output at the source.

Evidence: fix: never ship a placeholder image or empty-subject sentence and fix: stagger flow-diagram edge labels on main.
