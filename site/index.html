EVERY CLAIM CITED. EVERY CITATION CHECKED.

Glance turns a repository into a place you can walk through instead of a pile you have to read.

Glance recursively generates one rich HTML page per directory — leaf to root — narrating what each room is, how it fits the whole, and where it can hurt you. Every claim carries a file:line citation, hyperlinked to source at a pinned commit. After generation, a deterministic gate re-verifies every citation before the run can ship — a page with a broken citation does not go live. No LLM judges the output; the anchors are machine-checked.

Built from Aesthetic tokens, local assets, and GitHub Pages.

Why it matters

job Read the building, not a file listing — one page per directory, leaf to root. proof Every claim cites file:line; a gate re-checks it before the page ships.
shape A sister repo mirrors the source tree in generated HTML — nothing lands in the source repo. trust No LLM judges the gate. Citations are checked by code against the pinned SHA.
delivery Hash-gated: only changed directories regenerate. Cost scales with the diff. fit Depth-tiered models: cheap leaves, frontier root — a run reports its spend.
