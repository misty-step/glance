Get started

Glance is a Rust workspace. The commands below were verified against the current CLI surface and come from the repo README and SKILL.md.

Build the CLI

git clone https://github.com/misty-step/glance.git
cd glance
cargo build -p glance

Verified with cargo run -p glance -- --help.

First local run

cargo run -p glance -- plan --root /path/to/source --changed src/lib.rs
cargo run -p glance -- run --config glance.toml
cargo run -p glance -- check --source-root /path/to/source --source-sha <sha> site/index.html
cargo run -p glance -- serve-local --site-root site --port 4173

The tracked default config uses the deterministic mock provider so local gates and smoke runs do not depend on live secrets. Real provider mode is opt-in and uses environment credentials.

Publish deliberately

cargo run -p glance -- publish \
  --site-dir site \
  --source-owner misty-step \
  --source-name demo \
  --source-sha <sha> \
  --mode branch \
  --branch glance/source-pr-12 \
  --source-pr-title "Source PR title"

The SKILL.md contract defaults to branch mode unless the operator explicitly authorizes direct master publishing.
