# Self-run evidence, 2026-07-03

The first live glance-on-glance run generated 32 pages at
`~/artifacts/public/a/glance-self` from source SHA
`e68d722bd468c3ada09cb4ae8ce46e2429f8e4ca`.

The run proved the basic pipeline could walk the repository and write a site,
but it exposed three defects that made the output unsafe to trust:

- The root page at `~/artifacts/public/a/glance-self/index.html` was truncated
  at 1,608 bytes and ended mid-CSS instead of closing the HTML document.
  Its metadata shows `model=openai/gpt-5.5-20260423`,
  `input_tokens=50058`, `output_tokens=2600`, and `spend_micros=0`; the
  configured root cap was only 2,600 output tokens.
- Citation attributes drifted from the required `path:start-end` shape. Some
  generated attributes compressed multiple ranges under one path, such as
  `crates/glance-check/src/lib.rs:7-19,21-26,89-105`, which leaves `21-26` as
  a bare range after deterministic comma splitting.
- The spend report could show zero spend for a page with nonzero token usage.
  The root metadata recorded 50,058 input tokens and 2,600 output tokens with
  `spend_micros=0`, so budget and summary output understated the run.

The fixes from this evidence are: raise the default output caps, fail provider
output that reports token-cap termination or lacks a closing `</html>`, reject
malformed generated citations before writing pages, retry citation-format drift
once with deterministic feedback, and warn when a run reports tokens with zero
spend.
